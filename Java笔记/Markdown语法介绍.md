#什么是Markdown
Markdown 是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

#应用
1、用来写博客、邮件；
2、Markdown 很容易转化成 HTML 或者 PDF；
3、git 的 README.md 就是 Markdown 格式的。
